🚀 SkillForge LMS – Microservices-Based Online Learning Platform

SkillForge is a production-grade, scalable Learning Management System (LMS) inspired by platforms like Udemy. It is built using a microservices architecture to handle real-world scenarios such as course management, user enrollment, payments, reviews, and notifications.

🔧 Tech Stack:
- Frontend: Next.js (React), Tailwind CSS, Axios
- Backend: Spring Boot, Spring Cloud (Gateway, Eureka, Config Server)
- Security: JWT Authentication, OAuth2
- Databases: PostgreSQL, MongoDB, Redis
- Messaging: Apache Kafka (event-driven architecture)
- Search: ElasticSearch
- DevOps: Docker, Kubernetes
- Monitoring: Zipkin, Prometheus, Grafana

🎨 Frontend (Next.js):
- Server-side rendering (SSR) for better performance & SEO
- Dynamic course listing & filtering
- Secure authentication flow using JWT
- Instructor dashboard for course creation & management
- Student dashboard for enrolled courses & progress tracking
- API integration with microservices via API Gateway
- Optimized UI with responsive design (Tailwind CSS)

📦 Core Microservices:
- Auth Service (JWT-based authentication)
- User Service (students & instructors)
- Course Service (course & lesson management)
- Enrollment Service (student-course mapping)
- Order & Payment Service (Razorpay/Stripe integration)
- Review Service (MongoDB-based review system)
- Notification Service (Kafka-driven async notifications)
- Search Service (ElasticSearch powered search)

⚡ Key Features:
- Course creation & management (Instructor)
- Student enrollment & payment flow
- Event-driven architecture using Kafka
- Polyglot persistence (SQL + NoSQL)
- Scalable & containerized deployment
- Full-text search with ElasticSearch
- Review & rating system
- Modern responsive UI built with Next.js

📈 Why This Project?
This project demonstrates real-world full-stack engineering skills including microservices architecture, distributed systems, event-driven design, and cloud-native deployment with a modern frontend using Next.js.
